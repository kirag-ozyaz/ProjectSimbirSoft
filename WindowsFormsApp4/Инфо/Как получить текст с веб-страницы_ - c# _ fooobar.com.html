Как получить текст с веб-страницы?

Я не хочу получать HTML-код, я хочу, чтобы текст был удален с веб-страницы. Например, если я набираю "<b>cake</b>", я хочу торт, а не теги.

Используйте библиотеку HTML Agility Pack.

Эта очень тонкая библиотека для анализа HTML, для вашего требования используйте этот код:

    HtmlAgilityPack.HtmlWeb web = new HtmlAgilityPack.HtmlWeb();
    HtmlAgilityPack.HtmlDocument doc = web.Load("Yor Path(local,web)"); 
    var result=doc.DocumentNode.SelectNodes("//body//text()");//return HtmlCollectionNode
    foreach(var node in result)
    {
        string AchivedText=node.InnerText;//Your desire text
    }

Вы можете снимать теги с помощью регулярных выражений, таких как 2 (простой пример):

// You can import System.Text.RegularExpressions for convenience, of course.
System.Text.RegularExpressions.Regex tag = new System.Text.RegularExpressions.Regex("\<.+?\>");
myHTML = tag.Replace(myHTML, String.Empty);

Но если вам нужно извлечь большие объемы хорошо структурированных данных, вам может быть лучше использовать библиотеку HTML 1. (Если веб-страница - XHTML, тем лучше - используйте классы System.Xml.)

1 Например, http://htmlagilitypack.codeplex.com/.
2 Это может иметь непреднамеренные побочные эффекты, если вы пытаетесь получить данные из JavaScript или данные находятся внутри атрибута элемента и содержат угловые скобки. Вам также необходимо принять escape-последовательности, например &amp;.
